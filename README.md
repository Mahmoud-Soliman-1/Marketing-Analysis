# Marketing Analysis Project

## Project Overview
This project demonstrates an end-to-end marketing data analysis workflow using SQL, Python, and Power BI. The goal is to extract, process, and visualize marketing data to gain actionable insights into customer feedback, conversion trends, social media performance, and product ratings.

---

## Technologies Used
- **SQL**: Data extraction, transformation, and aggregation; calculated metrics such as average ratings.
- **Python**: Sentiment analysis to classify customer reviews and feedback.
- **Power BI**: Creation of interactive dashboards for visualization and analysis.

---

## Project Workflow

1. **Data Extraction & Transformation (SQL)**
   - Pull data from multiple tables and sources.
   - Clean and filter data to ensure accuracy.
   - Generate calculated fields (e.g., average ratings, conversion metrics).

2. **Sentiment Analysis (Python)**
   - Analyze textual customer reviews.
   - Classify feedback as positive, negative, or neutral.

3. **Visualization (Power BI)**
   - **Overview Dashboard**: High-level summary of marketing KPIs.
   - **Conversion Details**: Track conversion rates and trends over time.
   - **Social Media Insights**: Monitor social engagement and campaign impact.
   - **Rating Details**: Visualize average ratings and customer sentiment.

---

## Key Learnings
- Integrated multiple data sources into a coherent analytical pipeline.
- Applied sentiment analysis to extract meaningful insights from customer reviews.
- Developed interactive dashboards to communicate insights effectively.
